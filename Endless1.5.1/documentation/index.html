A. Getting started Back to top

Before we start, let's take a look on the folder structure of the theme. All theme files are placed in the HTML folder. The main HTML folder consists of 6 more folders for different types of theme assets:

1. Main Structure 

	HTML/                      		// Root directory
	    bootstrap/                     	// Twitter Bootstrap
	    fonts/                    		// Fonts
	    css/                           	// CSS Files
	    img/                           	// Default Images
	    js/                            	// Scripts
	    less/                          	// LESS Files

2. CSS Files

Endless Admin uses several of stylesheets to control visual apperance.The stylesheets consist of Twitter Bootstrap, default theme stylesheets and plugin stylesheets. 

	HTML/bootstrap/css/bootstrap.css		// Twitter Bootstrap Stylesheets
	HTML/css/endless.css				// Default Stylesheets
	HTML/css/endless-skin.css			// Skin Color
Note : There are more plugin-specific CSS files in the "css" folder. These CSS files are important for the correct work of the third party plugins.

3. LESS Files

LESS extends CSS with dynamic behavior such as variables, mixins, operations and functions.LESS runs on both the server-side (with Node.js and Rhino) and client-side (modern browsers only).

LESS files are located under the HTML/less directory.

ALL LESS variables are defined in variables.less.

Learn more about less css

4. Scripts

Endless Admin uses many third party plugins & scripts. There are scripts from Twitter Bootstrap, jQuery & plugins and custom scripts. They are all located in HTML/js/. The uncompressed js files are put in the HTML/js/uncompressed/ folder. For more information about external scripts refer to Credits.

B. HTML Structure Back to top

The main HTML structure of the template consists of 4 main containers containing all page elements:

  • Top navigation
  • Sidebar menu

1.2 Nesting columns

To nest your content with the default grid, add a new .row and set of .col-md-* columns within an existing .col-md-* column. Nested rows should include a set of columns that add up to 12.

Level 1: .col-md-9
Level 2: .col-md-6
Level 2: .col-md-6
